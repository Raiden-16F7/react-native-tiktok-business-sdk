# react-native-tiktok-business

A React Native bridge for the TikTok Business SDK

## Installation

```sh
npm install react-native-tiktok-business
```

## Usage


```js
import { multiply } from 'react-native-tiktok-business';

// ...

const result = await multiply(3, 7);
```
